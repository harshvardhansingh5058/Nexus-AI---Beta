🔥 Enable Real Google/GitHub Login

Supabase Authentication is free and takes about 5 minutes to set up.

Step 1 — Visit supabase.com → New Project → copy URL & anon key
Step 2 — Go to Authentication → Sign-in method → Enable Email/Password, Google, and GitHub
Step 3 (GitHub) — Create OAuth App at github.com/settings/developers
Callback URL: https://your-project.supabase.co/auth/v1/callback
Step 4 — Project Settings → API → Copy Project URL and anon public key
Step 5 — Open login.html and chat.html, replace YOUR_SUPABASE_URL and YOUR_SUPABASE_ANON_KEY
